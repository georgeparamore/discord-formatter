Select text, then click a button to wrap it. Line/block buttons apply to whichever lines your cursor or selection touches.
Quote All and masked [links](url) only render specially for bots/webhooks or inside embeds — in a normal user message Discord shows them as plain text.
ANSI colors only work inside a ```ansi code block, and only render on Discord's desktop and web apps — mobile shows the raw escape codes as plain text.
Discord has no image/video markdown — pasting a bare direct media link on its own line auto-embeds it when posted. Suppress Embed wraps a URL in <> to stop that preview from showing.
